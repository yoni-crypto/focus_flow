# FocusFlow

Productivity, finance, and focus unified in one minimal interface.

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS 4
- Shadcn/UI
- Supabase
- PWA

## Features

- Daily tasks with priorities and due times
- Pomodoro focus timer
- Money tracker with categories
- Notes with pin and archive
- User authentication

## Getting Started

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)

## Deploy

```bash
vercel
```
