# FocusFlow

A premium, minimal, mobile-first productivity, finance, and daily habit system.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS 4**
- **Shadcn/UI**
- **Supabase** (auth + database)
- **PWA** support
- **Vercel** hosting

## Design Philosophy

- Premium, minimal Vercel/Linear-style design
- Clean spacing, subtle borders, high contrast
- Beautiful typography, minimal icons
- Mobile-first layouts
- No gradients, no heavy shadows, no neon/glow

## Features

### Core Features (v1.0)

1. **Daily Planner** - Tasks with priorities, due times, swipe actions, drag & drop
2. **Focus Timer** - Pomodoro timer with session tracking and streaks
3. **Budget + Savings Tracker** - Daily spending/savings input with categories and charts
4. **Notes** - Quick note cards with pin/archive and search
5. **User Account** - Supabase auth with email/password

## Getting Started

First, install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
