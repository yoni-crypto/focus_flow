# Auth Background Image

## Image Requirements

Please add an image file named `auth-bg.jpg` to the `/public` folder with the following characteristics:

- **Format**: JPG or PNG
- **Recommended size**: 1920x1080 or larger
- **Content**: Dark aesthetic image featuring:
  - Large "FOCUS" text (white, bold)
  - Classical statue (dark/monochromatic)
  - Dark background (black/dark grey tones)
  - Minimal, elegant design

## Image Sources

You can find similar images on:
- Unsplash (search: "dark statue focus productivity")
- Pexels (search: "dark minimalist background")
- Create your own using design tools

## Alternative

If you don't have the exact image, the design will work with a solid dark background. The left section will display a dark gradient with the "FOCUS" text overlay.
