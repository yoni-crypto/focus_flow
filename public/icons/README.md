# PWA Icons

This directory contains the PWA icons for FocusFlow.

## Required Icons

The following icon sizes are required:
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Generating Icons

### Option 1: Using the Script

1. Create a `icon.png` file (512x512px) in the root directory
2. Install sharp: `npm install --save-dev sharp`
3. Run: `node scripts/generate-icons.js`

### Option 2: Online Tools

Use online PWA icon generators:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/

### Option 3: Manual Creation

Create icons manually using design tools:
- Use a simple, minimal design matching FocusFlow's aesthetic
- Background: Transparent or white
- Design: Minimal, recognizable symbol or letter "F"
- Ensure icons are clear at all sizes
